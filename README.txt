///////////////////////////////////////////////
///                                         ///
///              Requirements               ///
///                                         ///
///////////////////////////////////////////////

1. Built with Python 3.12. It will probably work with other versions.

2. ~10-20 minutes to run 

///////////////////////////////////////////////
///                                         ///
///               How To Use                ///
///                                         ///
///////////////////////////////////////////////

1. Run Install.bat

2. Run start.bat

3. Enter the username of the bar you would like to edit

4. The program will iterate through your item list, inactivating everything not in the previous Full Inventory
(NOTE) The full inventory has to be the 4th inventory in the list. We organize our inventories "Current Full/Recieving/Return/Previous Full"

///////////////////////////////////////////////
///                                         ///
///             ADDING NEW BARS             ///
///                                         ///
///////////////////////////////////////////////

1. Navigate to /barDB/

2. Open bardb.csv with excel or another spreadsheet software

3. Enter information for each bar you would like to work with

(NOTE)
user - Barkeep Username
pass- Barkeep Password
proper - The name of the bar
